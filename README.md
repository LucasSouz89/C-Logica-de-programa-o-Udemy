# 🚀 Lógica de Programação em C#

Bem-vindo ao meu repositório de Lógica de Programação! 

Este espaço foi criado para documentar o meu aprendizado e a minha evolução durante o curso de Lógica de Programação (Prof. Nelio Alves). Aqui estão centralizados todos os algoritmos, testes de mesa e exercícios práticos que forjaram a minha base como desenvolvedor.

## 🧠 O que foi desenvolvido e dominado:
- **Estruturas Sequenciais:** Manipulação de variáveis, entrada e saída de dados.
- **Estruturas Condicionais:** Fluxo lógico com `if`, `else` e `switch-case`.
- **Estruturas de Repetição:** Otimização de processamento com laços `while`, `do-while` e `for`.
- **Vetores (Arrays):** Armazenamento em memória, deslocamento de índices e estruturas de dados unidimensionais.
- **Matrizes Bidimensionais:** O grande desafio do curso. Manipulação de dados em eixos X e Y, leitura de tabelas na memória RAM e edição não-destrutiva de dados.

## 🎮 O Propósito (Próximos Passos)
A lógica estruturada documentada aqui é o alicerce da minha jornada. O objetivo principal é aplicar todo esse raciocínio matemático e de memória na **Orientação a Objetos (POO)** e, futuramente, dentro da **Unity**. Essa é a base do motor lógico por trás de projetos de Game Dev que estou desenvolvendo, como o *Equilibrium*, *Last Sentence* e sistemas de combate para jogos em Pixel Art 2D.
