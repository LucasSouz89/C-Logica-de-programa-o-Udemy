# 🚀 C# Learning Journey: Logic & OOP

🇺🇸 **English** | [🇧🇷 Português](#-português)

Welcome to my C# learning repository! This workspace contains all my exercises, notes, and projects developed during my studies of the C# language and the .NET framework, guided by Prof. Dr. Nélio Alves' courses.

My goal here is to document my evolution as a developer, transitioning from procedural problem-solving to robust structural software design.

### 📁 Repository Structure
The repository is divided into two main learning phases:

* **`/c# logica de programacão`**: Contains the foundation. Sequential structures, conditionals, loops (while/for), vectors, and matrices.
* **`/c# POO`**: Contains the core of software engineering. Classes, objects, attributes, methods, static members, encapsulation, inheritance, and polymorphism.

### 🛠️ Technologies
* C#
* .NET
