# 🚀 Programming Logic in C# / Lógica de Programação em C#

## 🇺🇸 English

Welcome to my Programming Logic repository! 

This space was created to document my learning and evolution during the Programming Logic course (Prof. Nelio Alves). Centralized here are all the algorithms, trace tables, and practical exercises that forged my foundation as a software developer.

### 🧠 What was developed and mastered:
- **Sequential Structures:** Variable manipulation, data input, and output.
- **Conditional Structures:** Logical flow using `if`, `else`, and `switch-case`.
- **Repetition Structures (Loops):** Processing optimization using `while`, `do-while`, and `for` loops.
- **Arrays (Vectors):** Memory storage, index shifting, and one-dimensional data structures.
- **Two-Dimensional Arrays (Matrices):** The biggest challenge of the course. Data manipulation on X and Y axes, reading tables in RAM, and non-destructive data editing.

### 🎮 The Purpose (Next Steps)
The structured logic documented here is the foundation of my journey. The main goal is to apply all this mathematical and memory reasoning to **Object-Oriented Programming (OOP)** and, later on, within the **Unity** engine. This is the core logic engine behind the Game Dev projects I am currently developing, such as *Equilibrium*, *Last Sentence*, and combat systems for 2D Pixel Art games.

### 🛠️ Technologies Used
- C#
- .NET 
- Visual Studio

## 🇧🇷 Português

Bem-vindo ao meu repositório de Lógica de Programação! 

Este espaço foi criado para documentar o meu aprendizado e a minha evolução durante o curso de Lógica de Programação (Prof. Nelio Alves). Aqui estão centralizados todos os algoritmos, testes de mesa e exercícios práticos que forjaram a minha base como desenvolvedor.

### 🧠 O que foi desenvolvido e dominado:
- **Estruturas Sequenciais:** Manipulação de variáveis, entrada e saída de dados.
- **Estruturas Condicionais:** Fluxo lógico com `if`, `else` e `switch-case`.
- **Estruturas de Repetição:** Otimização de processamento com laços `while`, `do-while` e `for`.
- **Vetores (Arrays):** Armazenamento em memória, deslocamento de índices e estruturas de dados unidimensionais.
- **Matrizes Bidimensionais:** O grande desafio do curso. Manipulação de dados em eixos X e Y, leitura de tabelas na memória RAM e edição não-destrutiva de dados.

### 🎮 O Propósito (Próximos Passos)
A lógica estruturada documentada aqui é o alicerce da minha jornada. O objetivo principal é aplicar todo esse raciocínio matemático e de memória na **Orientação a Objetos (POO)** e, futuramente, dentro da **Unity**. Essa é a base do motor lógico por trás de projetos de Game Dev que estou desenvolvendo, como o *Equilibrium*, *Last Sentence* e sistemas de combate para jogos em Pixel Art 2D.
